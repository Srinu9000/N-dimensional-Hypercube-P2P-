Run python Goal4.py in your respective python environment
1)Please make sure that you place init.py file and Goal4.py file are in the same directory.
2)Please enter the value of N(in between 2 & 11) to generate Hypercube with threads.
3)Enter the M value to decide no of files generated by each node.
4)Enter the P value which can vary from 1 to M
5) r- worktime of evil thread can be configured with code with q- number of nodes killed.
